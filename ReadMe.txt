---Solution----
The solution was developed using Visual Studio 2017 Community Edition on Windows 10 targeting .NET Core 2.1

Prerequisites:
- Any Terminal/Visual Studio Community Edition (Available for download at https://visualstudio.microsoft.com/)
- .NET Core 2.1

=====================================
To run unit tests
=====================================
- On any terminal move to the "FlattenArrayTest" folder (the folder containing the "FlattenArrayTest.csproj" file) and execute these commands:

dotnet restore
dotnet build
dotnet test

=====================================
To the program
=====================================
- On any terminal move to the "FlattenArray" folder (the folder containing the "FlattenArray.csproj" file) and execute these commands:

dotnet restore
dotnet build
dotnet run

=====================================
Special cases
=====================================
Enter exit on the command line to close the program
